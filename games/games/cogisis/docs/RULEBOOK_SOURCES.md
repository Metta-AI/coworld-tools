# Rulebook Sources

Primary rules source:

- Official Awaken Realms Nemesis rulebook PDF:
  https://awakenrealms.com/images/download/Nemesis/ENG/RULEBOOK_280x280mm_bleed3mm_28pages_PRINT.pdf
- Official Nemesis Games rulebook PDF mirror:
  https://www.nemesisgames.eu/documents/nemesis/Rulebook.pdf

Secondary reference used to cross-check high-level flow and endgame structure:

- UltraBoardGames Nemesis rules page:
  https://www.ultraboardgames.com/nemesis/game-rules.php

Implementation note: keep rule contracts paraphrased. The browser global viewer
uses a generated SVG ship layout informed by the official board structure rather
than copied rulebook images. Do not copy rulebook prose, card text, room sheets,
or component art unless that direction is explicitly requested.
