# Documentation

Current workspace documentation:

- `policy-catalog.md`: current source taxonomy and policy/tool locations.
- `source-provenance.md`: copied source commits, submodules, and excluded
  sources.
- `migrations/`: migration notes for source moves.
- `tutorials/`: tutorials copied into the new workspace layout.
- `experiments/`: experiment logs and research records.
- `bitworld/`: BitWorld player-policy notes copied from game repos.
- `audits/`: repo-structure and documentation audits.

Tool-specific documentation may live with its tool when that keeps code,
commands, and generated-artifact boundaries together. Cogbase documentation is
under `tools/cogbase/docs/` and is indexed from `tools/cogbase/README.md`.

Historical docs copied from the old `cogames-agents` package live under
`legacy/cogames-agents/`. Those files are retained as source material and may
still mention old `src/cogames_agents`, `scripts/`, or monorepo paths.
