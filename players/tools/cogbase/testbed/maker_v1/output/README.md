# maker_v1 Generated Artifacts

This directory is reserved for generated game-specific `maker_v1` outputs.

Files under `output/<game>/` are artifacts, not reusable toolkit code. They may
include generated agents, perception parsers, VLM frame labels, fixtures, test
harnesses, run logs, and submit-ready policy packages.

Do not edit files here when changing `maker_v1` itself. Change the generator in
`testbed/maker_v1/` and regenerate the relevant game artifact bundle.
